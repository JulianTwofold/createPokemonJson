JSON File mit allen Pokemon Daten erstellen

In diesem Auftrag geht es darum Daten von einer API (https://pokeapi.co/) zu holen und diese Daten schön darzustellen.

Ich möchte nur die Daten von Pokemon haben, die in dem aktuellsten pokemonspiel erhältlich sind. Ich könnte die API Mehrfach ausfragen, aber die Daten ändern sich nicht.

Deshalb macht es mehr Sinn alle Daten nur einmal zu holen und diese als JSON abzuspeichern. Wie ich das gemacht habe zeige ich auf dieser Seite schritt für Schritt.

Zuerst wollte ich die ID von jedem Pokemon haben, dass momentan erhältlich ist. Um das zu tun gehe ich auf die Pokedex Seite des aktuellsten Spieles (https://pokeapi.co/api/v2/pokedex/31).
